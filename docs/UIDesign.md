# UI设计原型文档

## 设计理念

- **简洁**: 界面清晰，信息层级分明
- **高效**: 减少操作步骤，提高效率
- **友好**: 易于理解，提供清晰的反馈
- **响应式**: 适配桌面和移动设备

## 设计系统

### 颜色方案

#### 主色调
```css
--primary-color: #1890ff;      /* 主色 - 蓝色 */

--error-color: #f5222d;        /* 错误 - 红色 */
```

```css
--text-primary: #000000d9;     /* 主要文本 */
```

## 交互设计

### 1. 语音输入流程

```
点击麦克风 → 请求权限 → 开始录音 → 显示波形
    ↓
停止录音 → 上传音频 → 显示加载 → 展示结果
    ↓
确认/重试
```

### 2. 创建计划流程

```
输入信息 → 验证数据 → 调用AI → 显示加载动画
    ↓
生成结果 → 预览计划 → 确认保存 → 跳转详情页
    ↓
编辑/分享
```

### 3. 编辑行程流程

```
点击编辑 → 拖拽排序/修改内容 → 实时保存
    ↓
添加新项 → 填写表单 → 保存 → 更新列表
    ↓
删除确认 → 执行删除 → 刷新视图
```

## 响应式设计

### 移动端适配 (< 768px)

- 隐藏侧边栏，改用底部导航
- 地图和行程垂直堆叠
- 卡片全宽显示
- 简化表单布局
- 增大触摸区域(最小44x44px)

### 平板适配 (768px - 1024px)

- 两栏布局
- 适度缩小间距
- 优化图表大小

### 桌面适配 (> 1024px)

- 三栏布局
- 固定侧边栏
- 充分利用空间

## 动画效果

### 1. 页面过渡
```css
.page-enter {
  opacity: 0;
  transform: translateY(20px);
}
.page-enter-active {
  opacity: 1;
  transform: translateY(0);
  transition: all 0.3s ease-in-out;
}
```

### 2. 加载动画
- 骨架屏 (Skeleton)
- 转圈加载器 (Spinner)
- 进度条

### 3. 反馈动画
- 成功: 绿色对勾动画
- 错误: 红色抖动
- 警告: 橙色脉冲
